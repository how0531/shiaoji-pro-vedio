Flash Order

閃電下單：點下去的瞬間，單已經出去了

價格梯自動跟隨現價，永不丟失行情。點價即下單、點單即刪。

  • 持倉損益常駐 — 部位、均價、未實現一眼掌握
  • 閃電全開 — 自選前 N 檔一鍵彈出、自動平鋪
  • 市價／平倉／全刪 一排快捷，Esc 鎖定

Grid Orders

鋪單：一次一排，跟著價格走

一鍵鋪出整排限價單，動態跟隨自動撤舊補新。

  • 組合單到價監控 — 合成價到位自動 IOC 出手
  • 停損停利／OCO — 進場自動掛保護
  • 風控中心 — 口數上限、虧損上限、一鍵全刪

Options

選擇權：損益曲線先畫，再下單

  • T 字報價 — ATM 自動定位、點擊連動下單

  • 帳務全景 — 多帳戶、權益保證金、一鍵平倉反手
