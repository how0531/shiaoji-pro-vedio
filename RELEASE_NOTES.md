## v0.1.25 — 伺服器管理翻修 + AI Agent 背景任務

### 伺服器管理（桌面版大翻修）
- **預設 port 8080 → 21322**（0x534A = "SJ"）：避開 8080 衝突；8080 仍是附掛候選，你自己跑的 CLI server 會被自動偵測接上，絕不誤殺。
- **探測式狀態管理**：伺服器狀態改用 HTTP 探測（上次使用/預設/舊 port），不再依賴 CLI state file——前景啟動的 server 之前完全查不到。
- **停止/回收更可靠**：以「port 佔用者 + 執行檔路徑證明」確認是本 app 打包的 sidecar 才停；app 結束時自動回收 sidecar（不再殭屍佔 port）；啟動前清掃遺失紀錄的孤兒 server。
- **備援 port 修正**：預設 port 被占走時 API base 會跟著 server 走，不再卡在「載入交易終端」。
- **sidecar 升級**：內建 shioaji server v1.5.3 → v1.5.5。

### AI Agent
- **對話中掛背景任務**：「開始幫我模擬交易」即可從對話建立 interval / daily / 到價 / 委託事件觸發的背景任務，可追蹤每次執行成效；自動下單權限的任務建立時一律跳核可卡，分析模式只允許唯讀任務。
- **觀察學習強化**：常駐觀察你的例行操作序列（重複 2–3 次才學），學到後主動告知；設定頁新增「操作觀察學習」開關（記錄只存本機）。
- **技能包全文注入**：已安裝技能包的 SKILL.md 全文進 system prompt，模型對 shioaji API 的認知從二手變一手；參考文件維持按需讀取。
- **SSE 斷流修正（正確性）**：串流中途斷線不再被當成正常收尾——之前可能讓 partial tool call 以空參數執行，現在偵測截斷、丟棄、自動重試，並加 90 秒 idle timeout（跑整天的背景任務不會永久卡死）。
- **shioaji CLI 一等公民**：agent 可用 CLI 做工具沒覆蓋的查詢（唯讀子命令直接放行）；CLI 下單/取消一律強制人工核可，不分權限。打包版 PATH 修正（Finder 啟動找得到 uv/homebrew 裝的工具）。
- **背景任務執行修正**：streaming 摘要不再重複累積、排程執行不再把自己誤判成重複任務、提案通知去重。

### 交易體驗
- **委託/成交回報完整可讀**：修正 SSE 事件解析（回報之前只剩「委託回報」「成交回報」標題）——現在委託成功/取消/改價/減量/失敗/成交都是完整訊息：商品、數量（口/張/股）、價格或市價、ROD/IOC、新倉/平倉/現股/融資券、委託書號、時間；失敗直接顯示原因與代碼。括號單的 SSE 成交快速路徑同時修復，成交當下立即掛出 OCO 停損停利。
- **K 線價格軸修正**：切換商品自動重新縮放（之前手動拖過價格軸後換商品，K 棒會跑出畫面）；工具列新增「重設視圖」按鈕。(#6)
