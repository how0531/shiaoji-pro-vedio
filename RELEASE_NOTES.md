## v0.1.32 — 指數即時行情修正

### 加權指數與基差
- 修正 Shioaji 1.7 `quote_idx` 的 PascalCase 欄位解析，加權指數、開高低、參考價、成交量與時間恢復即時更新。
- 基差改用完整解析後的 `TXFR1 target_code`；即使連續月合約先被搜尋或自選快取，也會正確映射實際近月事件。
- 行情訂閱首次失敗時不再誤標為已訂閱，server 重連或後續查詢可以重新建立訂閱。

### 指數介面
- 主報價板新增上漲、平盤、下跌、漲停、跌停與未成交家數，並支援指數即時閃動與 K 線尾端更新。
- 自選清單、選單列托盤與類股熱力圖全面接上指數 Quote；類股 snapshot 保留為初始與斷線備援。
- 沒有即時 Tick、五檔或下單能力的指數面板會顯示明確狀態，不再停在看似故障的空白畫面；歷史 K 棒、行情重播與回測仍可使用。

### 相容性與 Agent
- 舊指數代碼與舊 workspace pin（例如 `001`）會自動解析為正式 master code `IX0001`。
- AI Agent 查價與價格觸發支援 `quote_idx`，並在工具層直接阻止指數下單。

---

⚠ 回測結果基於歷史資料與簡化成本假設，不代表未來績效；AI 分析僅供參考；自動下單請自行評估風險，盈虧自負。
