# 停損靠意志力的日子，可能可以結束了

你有沒有算過，停損按慢一步的成本是多少？

某次急殺，從「想跑」到「成交」中間隔著：切視窗、打價格、按確認——那幾秒的滑價，比手續費貴多了。不是不會停損，是工具讓人慢半拍。

當沖或短波段做久了，這些你一定有感：下單流程太長，行情早跑了；停損靠手速和意志力，凹單往往就是這樣開始的；版面是死的，想要「兩張圖＋閃電下單」同屏，沒這選項。

最近永豐 Shioaji 官方 GitHub 上多了一個開源交易終端，把這些事一次處理掉了。

## Shioaji Pro 是什麼

一個涵蓋上市、上櫃、期交所的專業交易終端，掛在永豐 Shioaji 官方 GitHub 組織（Sinotrade/shioaji-pro-app）底下。介面 100% 開源（AGPL-3.0）——UI、行情串流、下單鏈路的程式碼全部攤在 repo 裡。就算你看不懂程式碼，重點是「任何人都看得到」，這跟黑盒子軟體的信任基礎完全不同。

架構也乾脆：沒有第三方雲端後端。前端直連你本機的 shioaji server，再經永豐官方 API 送單；API 金鑰在介面填寫、存在自己電腦的 App 資料夾。桌面版 macOS、Windows、Linux 都有安裝檔，內建 shioaji server，不需另外安裝 CLI，裝完填金鑰就能用。

先講前提：它是基於永豐 Shioaji API 的終端，需要永豐金證券帳戶，再到官網 API 管理頁建一組 API Key/Secret。

## 用一天的交易走一遍

**盤前** — 排行榜 scanner 用漲幅、量、額多條件複選（含放空篩選），配類股熱力圖看今天資金往哪堆；候選標的順手看籌碼卡——融資券、借券，處置股先剔掉，省得盤中才發現撮合方式不對。

**開盤** — 閃電下單價格梯，左欄買、右欄賣，點下去就是單（預設有安全鎖防誤觸）。進場搭括號單：成交那一刻自動掛好 OCO 停損停利，出場計畫先綁死，不留凹單空間。

**盤中** — 未成交委託在 K 線圖上是一條實線，改價直接拖，不用刪單重掛；停損停利用虛線畫在圖上，隨時可取消。行情不對勁，Esc 連按兩下，全部刪單。細節也講究：自選清單只在「真實成交」才閃動，試撮不閃，尾盤不會被假動作騙。

**收盤後** — 行情回放用 1x–100x 變速重播當日 tick，把自己猶豫、追高、砍在阿呆谷的那幾秒重看一遍，比任何檢討文都有感。

## 給上頭的自己一個開關

當沖做到失控時，你需要的不是更多訊號，是能把自己關掉的按鈕。風控 Kill Switch 可以設單筆上限、日虧上限、一鍵鎖單；下單面板是兩段式確認，按第一下確認、第二下才送單；斷線時自動鎖定下單按鍵，重連後自動重新訂閱行情。

規格面該有的都有：整股/零股、ROD/IOC/FOK、期貨倉別與沖賣，下單前預估手續費和稅；21 種內建技術指標，不夠用可以直接在面板裡用 JavaScript 寫自己的；選擇權有 TXO T 字報價、到期損益圖和組合單。版面全部拖拉自訂、面板可彈出獨立視窗丟第二螢幕。

## 先講醜話

1. 需要永豐金證券帳戶，沒有就用不了。
2. 停損停利是「客戶端觸價單」，由 App 在你電腦上監控——程式關掉就不會觸發。掛完單想蓋電腦出門，請先想清楚。
3. 預設是模擬環境（假錢模式），頂部有徽章提醒；切正式環境要先設 CA 憑證、徽章會變紅——之後每一筆委託都是真錢，風險自負。
4. 這是新成立的專案，還在快速迭代；AI Agent 和策略回測是桌面版專屬（閉源）模組，不在開源 repo 裡。

## 三步驟上手

1. 到 GitHub Releases 下載對應平台安裝檔：https://github.com/Sinotrade/shioaji-pro-app/releases
2. 在永豐官網 API 管理頁建立 API Key/Secret，填進 App（金鑰只存本機）
3. 留在模擬環境，把閃電下單、括號單、刪改單整套流程用假錢跑熟，再考慮切正式環境

官網與文件：https://sinotrade.github.io/shioaji-pro-app/

工具不會讓你變成穩定獲利的交易者，但慢半拍的工具，確實會讓你少賺多賠。先用假錢把恐懼練掉——反正介面的程式碼都攤在那裡，不用聽誰的業配，自己驗證最快。
