## 進6 ｜ 訂單與委託：怎麼選、怎麼看（純可做 · 三族群底層必修）
mode: app-2317（開場跳 2317）

| 段 | 動作 | 旁白 |
|---|---|---|
| o-1 | 下單面板總覽（zoom 面板） | 這集把「該選哪種單、怎麼看委託」講清楚。先看下單面板：上面選買賣，中間填價量，下面是價別、效期、單位。選項很多，但你只要懂幾個關鍵。 |
| o-2 | 切「限價 LMT」再切「市價 MKT」示範 | 先講價別。限價，就是指定一個價格，市場走到那個價才成交——好處是價格你說了算，壞處是可能一直不成交。市價，是不管價格、立刻成交——好處是一定買得到賣得掉，壞處是急殺急拉時，可能成交在很差的價位。新手建議先用限價，比較安全。 |
| o-3 | 切 ROD／IOC／FOK 示範 | 再看效期。ROD 當日有效，掛著等到收盤，最常用。IOC，立即成交、沒成交的部分馬上取消。FOK，要嘛全部成交、要嘛整筆取消。一般掛單用 ROD 就好，IOC、FOK 是搶量或大單控制才用。 |
| o-4 | 切整股／零股、（股票）現股當沖列 | 單位這裡，可以切整股或零股。零股就是不滿一張、一股一股買，小資很常用。做股票當沖的話，這裡還會有現股當沖的選項，先賣後買或先買後賣——這風險比較高，確定會當天回補再用。 |
| o-5 | 決策卡（靜態卡） | 一句話幫你選：怕買在爛價位就用限價；要確定成交才用市價。想長抱、分批進出就 ROD 慢慢掛；要搶要快才用 IOC、FOK。資金小先玩零股。當沖、融資這種有槓桿的，等你很熟再碰。 |
| o-6 | 委託分頁＋狀態卡（動作＋靜態卡） | 送出後怎麼看單？到委託分頁。這裡最重要的是「狀態」：委託成功，是券商收到了、但還沒成交；成交，才是真的買到賣到；部分成交，一張單只成交一部分，剩下的還掛著；廢單，代表這筆被退掉、沒生效。 |
| o-7 | 廢單原因卡（靜態卡） | 為什麼會變廢單？最常見幾個：掛的價格超過當天漲跌停、零股規則沒對、融資融券額度不足、或是價格不符跳動單位。看到廢單別慌，看一下原因、改對了重掛就好。這集就到這裡。 |

## 進7 ｜ 帳務與交割安全：別讓自己違約（安全必修 · newbie+swing 雙 must）
mode: app（開場 2330；主要在持倉/委託/帳務面板）

| 段 | 動作 | 旁白 |
|---|---|---|
| a-1 | 開帳務分頁總覽 | 這集講最容易被忽略、但出事最嚴重的：帳務與交割。打開持倉委託帳務面板，切到帳務分頁。這裡是你帳戶的水位表。 |
| a-2 | 帶到餘額/股票市值/資產市值 | 上面幾張卡：證券交割帳戶餘額，是你可動用的錢；股票市值，是持股現值；資產市值，是全部加總。買股票前，先確認交割帳戶有沒有足夠的錢。 |
| a-3 | 交割款觀念卡（靜態卡） | 台股是「T+2 交割」——你今天買，錢是後天早上才從交割帳戶扣。所以重點來了：買進的當下帳戶可以不用有錢，但後天早上一定要有，不然就是違約交割。 |
| a-4 | 違約後果卡（靜態卡） | 違約交割不是小事：會被券商追繳、通報聯徵、影響你的信用，嚴重還有法律責任。所以資金小的人，永遠只買「後天付得出來」的金額，這是鐵律。帳務分頁的近日交割款，就是幫你看未來幾天要付或會收多少。 |
| a-5 | 帶到期貨權益數/保證金/風險指標 | 如果你做期貨，這裡還有權益數、可用保證金、還有風險指標。風險指標低於一定水準會示警——低了代表快被追繳、甚至被強制平倉，看到它變色就要補錢或減倉。 |
| a-6 | 融資維持率觀念卡（靜態卡） | 用融資買股票的話，要盯「維持率」。維持率就是你的擔保品價值除以借款，跌太多會收到追繳、再不補就被斷頭賣掉。用槓桿前，先想清楚跌到哪你會被追繳、你補不補得起。 |
| a-7 | 資產分布甜甜圈＋前五大 | 最後，帳務分頁下面有資產分布圓餅和前五大持股，一眼看出你的錢集中在哪。太集中在單一股票或單一類股，風險就高，該分散。這集就到這裡。 |

## 進8 ｜ 一鏡到底：一筆完整交易（#1 共識 · 整合實戰）
mode: app-2317　※涉及成交/平倉的以「盤中的時候…」條件式帶過（非交易日委託會秒撤）

| 段 | 動作 | 旁白 |
|---|---|---|
| w-1 | 排行複選挑一檔（或自選點一檔） | 前面每一集都在教單一功能，這集把它們串成一筆完整交易，從頭走到尾。第一步，選股。用排行榜複選，篩出量價都夠的一檔，點下去，整個畫面連動過來。 |
| w-2 | 帶到現價＋口述算量 | 第二步，算你買得起多少。假設你有二十萬，這檔現價假設一百塊，一張是一千股就要十萬，你買得起兩張；買不起整張，就用零股，二十萬可以買大約兩千股。先算清楚，別下超過你付得出的交割金額。 |
| w-3 | 下單面板填限價、切零股、兩段式送出 | 第三步，下單。我用限價、切到零股，填好數量，點買進、再點確認，兩段式送出。我們在模擬環境，這筆是假錢。 |
| w-4 | 切委託分頁看狀態 | 第四步，確認。到委託分頁看狀態——委託成功代表券商收到了；盤中成交了，狀態才會變成成交。看不到成交別急著重下，先確認是還沒輪到你的價，還是被退成廢單。 |
| w-5 | K線切停損模式（展示，不觸發） | 第五步，先想好出場。切到停損模式，盤中你可以在圖上點一個價位掛停損，跌到就自動出場。進場的同時就把停損想好，這是活得久的關鍵。 |
| w-6 | 平倉觀念＋帳務含費卡（靜態卡） | 第六步，平倉結算。盤中要賣的時候，一樣在下單面板賣出、或用持倉一鍵平倉。真正賺多少，要看「含手續費和證交稅」之後的淨損益——買賣各一趟手續費、賣出還有證交稅，本金小的時候，這些成本很容易把小賺吃掉，所以一筆要賺得夠多才划算。 |
| w-7 | 收尾卡 | 選股、算量、下單、看回報、設停損、平倉看淨賺賠——這就是一筆完整交易。先在模擬環境把這一輪走熟，再用真錢下場。 |

## 產出注意（給製作 agent）
- 全 2K；每段旁白↔動作成對，逐段對齊自檢＋獨立檢核。
- 靜態卡（決策卡/觀念卡/廢單原因卡）用 generate-cards 產（永豐配色、雅黑粗標/正黑），當該段畫面。
- 帳務數字非交易日可能為 0/空 → 旁白描述「欄位在哪、代表什麼」，不宣稱具體數字。
- 一鏡到底涉及成交/平倉一律條件式（盤中的時候…），委託只送出不假演成交。
- 若某欄位（如股票融資維持率）App 未提供，改用觀念卡帶過，不硬指畫面。
