### 旁白連續稿（配音用）

切出去下單、再切回來看盤，漏掉多少行情，你算過嗎。這章用一分鐘，先認識 Shioaji Pro。它是專業的交易軟體，做台股、期貨、選擇權。背後是永豐 Shioaji API，也就是永豐證券的行情和下單通道。看盤、行情、下單這條主線完全開源，程式碼都攤在 GitHub 上，誰都能查。但記著，AI Agent 和策略回測不公開程式碼，只有桌面版才有。給誰用。做當沖的，就是當天買、當天賣。盯選擇權的。愛在多個價位掛一排單的。一句話，為盤中要快、要準的人設計。核心價值有四個。第一，看圖下單。在 K 線圖上點個價位，單就掛出去了。第二，閃電下單。價格梯把價格由高到低排成一排，點哪個價就掛哪個價。第三，風控。一個鍵就能把下單整個鎖住。Esc 連按兩下，掛著的單全部刪掉。第四，版面自訂。整個畫面像一面牆，面板隨你拼，也能一鍵套內建版型。它有桌面版，也能直接開 Web 版。最安心的是，它預設就是模擬環境，用的是假錢。放心把每個功能點過一輪，練熟了再說。這章先到這裡。下一章實際動手。帶你申請 API 金鑰、填進去、完成登入。API 金鑰就像一組通行證，讓軟體能用你的帳戶跟券商溝通。然後把各項功能一個一個走過。

上一章我們大概認識了畫面。這一章要動手做一件事，就是把這個軟體真正連上永豐，也就是完成 API 登入設定。先解釋一下。API 金鑰就像一組通行證，有了它，這個軟體才能用你的帳戶跟券商溝通。第一次打開 App，畫面會自動跳出一個「設定精靈」，帶你把資料填一填。先提醒一件事。這個設定精靈只有桌面版 App 才有。開源的 Web 版沒有本機伺服器，不會出現這個精靈，也沒有等一下會講到的伺服器按鈕。整個流程就三步。第一步，去永豐申請一組金鑰。第二步，把金鑰填進設定精靈。第三步，選好環境、按下啟動，就能進到交易終端。我們一步一步來。首先要拿到金鑰。請注意，這一步你要先有一個永豐金證券的帳戶才行。登入永豐的「API 管理頁」，在裡面申請一組 API Key 和 Secret Key。這兩把就是你的通行證，等一下要填進 App。補充一下。桌面版畫面右側可能會出現一張「AI 助理 — 引導申請 API Key」的卡片，可以帶著你走申請流程。不過這是桌面版才有的功能。如果你的版本沒有這張卡片，就自己到永豐的 API 管理頁申請就好。拿到金鑰，我們回到 App。第一次啟動會自動停在設定精靈這一頁，你不用自己去找。如果你手邊已經有存好金鑰的 .env 檔，可以點上方的「選資料夾自動讀取 .env」。小提醒，它要你選的是「資料夾」，不是檔案本身。選好之後，它會自己進去找 .env，幫你把金鑰填好。你也可以手動貼。在「API KEY」和「SECRET KEY」兩個格子，分別貼上金鑰。貼完看一下每個格子右邊的眼睛按鈕。點一下，就能把遮起來的金鑰變成看得見的字，方便你檢查有沒有多打空白、有沒有少字。接下來選環境。「環境」這一列有兩顆按鈕，模擬環境和正式環境，預設就是模擬。我強烈建議你先待在模擬環境。別擔心，模擬環境是練習用的，裡面是假錢，下單不會動到你的真錢。你可以放心把整套操作練到熟。所以第一次設定，這裡別動，保持模擬就好。那哪天想真的下單呢。我先示範給你看，但請等你練熟再切。點下「正式環境」，畫面馬上跳出紅字警告。上面寫著，正式環境下單動用真實資金，而且需要 Sinopac.pfx 憑證。意思就是，切過去之後，每一筆都是真金白銀。切到正式後，下面會多出一個「憑證」區塊。先點「選擇 Sinopac.pfx…」，挑出你的憑證檔。接著下面會出現「憑證密碼」欄，填你當初下載憑證時設定的密碼。金鑰、憑證、密碼，三樣缺一不可。少了任何一樣，系統在啟動前就會擋下來提醒你。示範完了，我把它切回模擬環境。金鑰填好、環境選好，就點最下面的「啟動並開始使用」。畫面會跑一條進度動畫，並提示「登入與載入合約約需 10 到 30 秒」。它在背後幫你登入永豐、把合約資料載進來，稍等一下就好。如果金鑰或憑證有問題，它會用看得懂的訊息告訴你哪裡錯。一切正常的話，等伺服器回報健康，頁面就會自動重新載入，帶你進到真正的交易終端。最後補充兩件事。第一，這些設定以後不用再跑一次精靈。進到終端後，你看畫面頂部的右上角，有一顆帶狀態燈、寫著「伺服器」的按鈕。點開就是「伺服器管理」。想改金鑰、想在模擬和正式之間切換、想換憑證，都在這裡改。提醒一下，環境或憑證改完，要按「重啟」才會真的生效。這顆伺服器按鈕跟設定精靈一樣，只有桌面版 App 才有，Web 版沒有本機伺服器，看不到這顆按鈕。裡面還有一顆「下單就緒檢查」。以後如果正式環境下單失敗，按它就能一鍵檢查帳戶和憑證是哪裡沒過。第二，也是最重要的。你填的金鑰只會存在你自己這台電腦，不會上傳到任何地方。而且務必先在模擬環境練熟，再考慮切到正式。這一章就到這裡。下一章我們正式開始看盤。

### 錄製注意

整章務必在桌面版(Tauri App)實機錄製,因為首次設定精靈與伺服器管理面板都是桌面版專屬,Web 版根本錄不到這些畫面;且整章請全程用『模擬環境』示範,不要真的用正式環境與真金鑰。Tauri 桌面 App 不是純瀏覽器頁面,Playwright 難以自動驅動與錄製,建議 2-1、2-3、2-4、2-5、2-6、2-7 都以桌面螢幕錄影(OBS 之類)為主,2-2 另切瀏覽器錄。

- 2-1、2-3(輸入金鑰畫面,建議手動螢幕錄影):在設定精靈輸入 API KEY／SECRET KEY 一律用假金鑰(例如 fake_key_xxx)示範,絕不露出真實金鑰;點眼睛鈕切明碼那一下要再確認畫面顯示的是假字。錄完檢查有無殘影、剪貼簿浮窗或輸入法候選字露出真值。
- 2-2(手動螢幕錄影+後製遮罩):切到瀏覽器錄永豐『API 管理頁』,實際金鑰、帳號、姓名等個資一律打馬賽克或用假字覆蓋;建議先錄好再上遮罩,不要直接對外串流。
- 2-4、2-5(可直接錄):環境切換與正式環境紅字警告可直接錄。2-5 特別要照新的『兩步時序』錄:先切正式環境 → 憑證區塊此時只出現『選擇 Sinopac.pfx…』按鈕 → 點按鈕選好憑證檔後,下方才出現『憑證密碼』欄位,不要讓兩者一開始就同時出現。憑證密碼欄即使示範也填假密碼,別填真密碼;錄完務必切回模擬環境。
- 2-6(可直接錄,注意會真連線):按『啟動並開始使用』會實際去登入永豐並載入合約(約 10–30 秒),用模擬環境金鑰跑即可,錄下進度動畫到自動進入終端。若不想每次真連線,可先跑好一次、之後只補錄畫面片段。
- 2-7(可直接錄):伺服器管理浮層可直接錄,但『API 金鑰』區欄位一樣要顯示假字;錄到『下單就緒檢查』按鈕帶過即可,不必真的對正式帳戶做檢查。
- 2-1 與 2-7 記得依本次修訂,在畫面角落疊一行『桌面版 Tauri App 專屬』的小字註記字幕,對應這次補上的 caveat,避免觀眾誤以為 Web 版也走同一套登入/伺服器流程。

### 旁白連續稿（配音用）

上一章我們已經登入，也把版面擺好了。這一章帶你把主要功能走一遍，分成四組：看盤、下單、選股，還有選擇權。先從看盤的起點講起：自選清單，就是你關注的商品名單。你在這裡點哪一檔，開著「連動」的面板就會跟著跳過去：點台積電，K 線圖、五檔、下單面板全部跟著換。想加商品，在面板最下面的輸入框打代碼或名字，跳出建議後點一筆就加進來。再來看 K 線圖。左上角有五顆週期鈕：一分、五分、十五分、六十分、日線，點一下就切換。想看更早的走勢，把圖往左拖，系統會自動接上更舊的資料，最遠能拉回大約三年前。畫面拖亂了別慌，點週期鈕右邊的「重設視圖」，K 棒馬上回正。右邊是五檔，就是市場上最靠近成交價的五個買價和賣價。它最好用的是「點價帶入」：點任一檔的價格色條，那個價格就自動填進下單面板。放心，它只是填價格，不會直接送單；真正下單，還是要在下單面板確認。想看更細，可以搭配兩張表。成交明細把每一筆成交列出來，最新的在最上面，時間細到微秒。分價量表把今天的成交量照價位堆成橫條，頂部還有內外盤比，就是主動買和主動賣的力道比例。收盤後想練盤感，用行情回放。它把今天的成交重播一遍：按播放，價格線就往右跑。速度有一倍、五倍、二十倍、一百倍四段，也能拖進度條跳到想看的那段，反覆看一個轉折怎麼走出來。看盤的部分到這裡，接下來講怎麼下單。先看下單面板。整股、零股、期貨、選擇權，全用同一張表單，零股就是不滿一張的股票。你選什麼商品，欄位就自動跟著換。它最重要的設計是兩段式確認：第一次點「買進下單」，單子不會送出去，按鈕會先變成「確認買進幾張、什麼價格」讓你核對，再點一次才真的送出。而且每筆送出前，系統都會先做風控檢查。短線當沖可以用閃電下單。它是一條直的價格梯：中間是價格，左邊買量，右邊賣量。注意它預設是鎖住的，這是最重要的安全設計。先點「啟用閃電下單」，按鈕亮成「點價即下單」才會生效。之後就是左買右賣：點左邊掛限價買、點右邊掛限價賣，限價就是指定價格、到價才成交。提醒你，這是真的委託，點一下就送出去。我們示範用的是模擬環境的假錢；換到正式環境，就是真錢真單。想收手，隨時按 Esc，面板馬上重新鎖上。還有兩張進階面板。鋪單讓你在現價上方或下方，一次掛出一整排限價單：設好從哪開始、掛幾檔、間隔多少，解鎖後一鍵送出。組合單是給兩隻腳同時進場的策略用的，像期貨換月、選擇權價差或跨式。你輸入兩腳合約，它會算出整組的合成報價，還會自動判斷這是哪種組合。再來是守紀律的工具：頂部的「風控」選單。點開最上面紅色的「鎖定下單 Kill Switch」，它像一個總開關，按下去，一般委託全部當場擋下，是情緒上頭或行情異常時最快的剎車。不過它只管不讓新單出去。已經掛出去、還沒成交的單，就在零點六秒內連按兩次 Esc，全部刪掉。停損停利則在下單面板打開「停損停利保護」，填好停損價和停利價，進場成交後系統自動掛一組 OCO 保護單。OCO 就是兩張單綁在一起，一張成交，另一張自動取消。但有一件事一定要記住：這組保護單，還有圖上設的觸價，都是這台電腦、這個 App 在幫你盯的。只有程式開著、行情連線正常，它才有效。你一關掉程式，它就不會幫你觸發了。所以重要的停損，千萬不要完全交給它。下單講完了，接下來看怎麼選股。先看排行榜。它可以照漲幅、跌幅、量、金額排序，但更好用的是「複選」：同時設漲幅、量、金額三個門檻，一次篩出量價都夠、真的能操作的名單，也能切放空模式，反過來篩弱勢股。點任一列，整個終端就連動到那檔。注意，這只是連動過去看，不會加進自選清單。搭配類股熱力圖，就是由上往下選股。第一層是類股總覽，顏色代表各類股今天的漲跌，最強的排最前面。點進一個強勢類股，就鑽到第二層看裡面的個股，點個股一樣連動整個終端。想回上一層，點左上角的「總覽」鈕就好。下單前想確認風險，看個股籌碼卡。它把重點集中成一張卡：能不能融資融券、有沒有券可借、有沒有被處置警示、能不能當沖。進場前掃一眼，就知道這檔有沒有交易限制。這張卡只支援上市櫃個股。只做股票的朋友，接下來這段講選擇權，可以先跳過。選擇權用 T 字報價：買權排左邊、賣權排右邊，CALL 是買權、PUT 是賣權；中間那欄是履約價，就是約定的買賣價格。整張表以台指期現在的價格當中心，往上下展開履約價。點一列的左半邊連動買權，點右半邊連動賣權，而且點的那隻腳會同時丟給組合單。所以先點一個 CALL、再點一個 PUT，組合單的兩隻腳就湊齊了。最後是損益圖。它把你的選擇權部位，畫成一條到期結算的損益曲線，標出最多賺多少、最多賠多少，也可以在底部自己加模擬腳試算。注意工具列的警語：它畫的是到期那天的結算結果，沒算時間價值，也沒算手續費，只能當參考。四大功能組導覽到這裡：看盤、下單、選股、選擇權。每個面板都能自由搬動、縮放，也能連動或鎖定。下一章，我們先把版面系統和面板細節走熟；到第五章，再帶你從選股到下單，完整走一遍。

### 旁白連續稿（配音用）

前面幾章認識了各個面板，這一章來講版面系統，還有幾個一定要知道的細節。整個工作區不是固定死的畫面，比較像一面可以自由拼貼的牆，每個功能都是牆上的一塊板子。先講怎麼加新面板。入口只有一個，就是頂部右邊的『＋ 新增面板』。點開它，裡面列了全部二十四種面板，點哪種，就長在工作區最下面。不能用拖的，也沒有右鍵選單。注意，自選清單、排行榜這種面板只能開一個。開過了，選單那一項就變灰、標『已存在』，點不動。再來是搬家和改大小。每個面板最上面那條標題列就是把手，按住往任何方向拖就能搬走，其他面板會自動讓位。別擔心誤觸，按鈕和輸入框不算把手，你點鈕、打字不會拖走面板。想改大小，游標移到面板邊緣或右下角按住拖。每種面板都有最小尺寸，縮到底就不會再小。接下來這顆鈕最關鍵，叫連動和鎖定。只要是會顯示某一檔商品的面板，標題列上都有它。預設是連動，意思是你在別處點哪一檔，它就跟著跳到哪一檔。點它一下，圖示變成圖釘，面板就被釘住，只顯示現在這一檔，你在別處怎麼切都不影響它。鎖定後標題列還會多一個代碼框，直接輸入代碼、按 Enter，就換成那一檔。再點一下鈕，回到連動。自選清單、排行榜本來就是拿來選商品的源頭，所以沒有這顆鈕。如果你有兩台螢幕，還有一招。部分面板標題列上有一顆『彈出』鈕，點下去，面板就變成獨立視窗，整個拖到第二台螢幕專心盯。一共十一種面板支援彈出，K 線圖、五檔、下單、閃電下單這些主力都在裡面。懶得從零擺也沒關係。頂部『版面』選單裡有一整排現成版型，像標準看盤、當沖交易、雙圖對照、閃電矩陣、熱力選股。滑鼠停在名字上會顯示用途，點下去整組直接套上，蓋掉目前的畫面。自己擺好之後，一樣在『版面』選單打個名字存起來，之後一鍵載入，不要了也能刪。這些版面只存在你自己的電腦裡，不會上傳。平常每拖一下、每加減一塊面板，系統也馬上存回本機，下次打開自動還原。老實提醒一件事。『AI 副駕』版型用到的 AI Agent 和策略回測，是桌面版才有的功能，沒有開源。用開源 Web 版套這種版型，那塊面板只會顯示鎖定畫面。版面講完，來看細節。先看 K 線圖，它不只能看圖，還能直接在圖上下單。工具列中間有六種模式，游標、點價買、點價賣、停損、停利、警示。切到『點價買』，在圖上想買的價位點一下，就用那個價位掛出限價買單。限價單就是指定價格、等市場走到才成交。但注意，點一下，單就真的送出去了，不會再跳任何確認視窗。動手前先看清楚模式，還有旁邊的『量』設對了沒。另外除了游標，每種模式點完一筆就自動跳回游標。你掛出去還沒成交的單，都用實線畫在圖上。想改價，游標移到線附近會變成上下箭頭，按住拖到新價位放開，系統就幫你改，成功會跳『改價已送出』。想取消，點圖左上角清單那筆的『CANCEL』就好。停損、停利、警示則用虛線畫，你點在現價上面或下面，方向它自動判斷。每條虛線後面有個小叉，點了就移除。接下來這段一定要聽進去。圖上掛的停損、停利、警示，是這個軟體在你電腦上自己盯著行情幫你看的。它們存在瀏覽器裡，重新整理不會不見。但只有 App 開著、行情連線正常時才會動作。一關掉程式或斷線，就沒人在盯，價格到了也不會觸發。所以真正重要的停損，請改用券商伺服器端的委託。再來看閃電下單，這是給短線用的價格梯，一排價位列在眼前，點哪個價位就用那個價位下單。它最重要的安全設計是預設上鎖。控制列那顆鈕鎖著時寫『啟用閃電下單』，點一下才亮成『點價即下單』，梯上的格子才有作用。解鎖後規則很簡單，左買右賣。點左邊的買量欄掛限價買，點右邊的賣量欄掛限價賣。任何時候按 Esc 馬上重新上鎖，行情一斷線也自動退回鎖定。掛出去的單刪起來也快。哪個價位有還沒成交的單，那一列就出現一顆小標籤，顯示剩幾口，點它就把那個價位、那個方向排隊中的單全撤掉。控制列下面還有市價買、市價賣，有部位時多一顆『平倉』，一鍵反向出掉。最右邊『全刪』，把這檔所有沒成交的單全部撤光。想一次盯一整排，頂部『⚡ 全開』選單能把自選前幾檔各自開成獨立的閃電視窗。最後回到下單面板，補兩個細節。第一個是價別和效期。價別就是用什麼價格下單，股票有限價和市價，期貨多一種叫 MKP 的市價單。效期是這張單有效多久。ROD，掛到收盤前都有效。IOC，能成交多少算多少，剩下馬上取消。FOK，要嘛全部成交，要嘛整筆取消。選到市價類，價格欄會鎖住，效期自動切成 IOC。切回限價，自動回 ### 錄製注意

全程務必在『模擬環境』錄製（頂部環境徽章顯示『模擬環境』），因為本章大量示範會送出真實委託流程的動作（點價買賣、閃電點價、兩段式確認），切勿用正式環境（真錢）錄。開場可打一張『本影片畫面皆為模擬環境示範』字卡。

【可用 Playwright 自動錄的段落】版面操作為主：seg 4-02 展開『＋ 新增面板』下拉並示範『（已存在）』灰項、seg 4-03 拖曳標題列與拖邊角縮放（用 mouse down/move/up 對 drag-handle 派事件；resize 抓面板右下角 handle）、seg 4-04 連動↔鎖定切換與鎖定後輸代碼、seg 4-06 版面選單套用預設、seg 4-07 命名儲存/載入。這些純前端互動可穩定腳本化。

【建議手動螢幕錄影（OBS 全桌面）的段落】凡涉及『彈出獨立作業系統視窗』的都建議手動錄：seg 4-05（單一面板彈出到第二螢幕）與 seg 4-12 的『⚡ 全開』（一次開多張獨立閃電視窗）——這會開出 OS 層新視窗、跨螢幕，Playwright 的 popup/多視窗處理較不穩，用 OBS 錄含第二螢幕的整個桌面最保險。

【下單相關示範】seg 4-08~4-10（K 線圖點價下單／拖委託線改價／虛線觸價）、seg 4-11~4-12（閃電解鎖點價、chip 刪單）、seg 4-13（下單面板兩段式確認）都在模擬環境操作；可自動或手動，但點價買/賣、市價買/賣會真的走送單流程，錄前先確認徽章為模擬環境。seg 4-13 記得特寫按鈕文字從『買進下單』變成『確認買進 N 張 @ 價』的兩段變化。seg 4-10 的『客戶端監控／關 App 即失效』與可彈出面板全名，用後製字卡呈現最清楚。

【隱私提醒】畫面會帶到帳號、部位、損益數字：錄影前先到頂部『主題』選單開啟隱私遮蔽（帳號、名稱、金額遮蔽）或使用模擬帳號，避免真實帳務入鏡。本章不涉及輸入 API 金鑰的畫面；若剪接時順帶帶到登入或伺服器管理頁，Key/Secret 一律不入鏡（金鑰只存本機），該畫面務必手動錄並打碼。

### 旁白連續稿（配音用）

最後，把學過的串起來，實戰走一遍。先選股。打開排行榜的「複選」，用漲幅、成交量、成交金額三個門檻，篩出有漲、量又夠的名單。看到順眼的，點下去。整個畫面跳到這一檔，K 線、五檔、下單面板全部到位。快速看兩眼。K 線看走勢位置。五檔就是買賣雙方掛的前五個價位，誰掛得多，力道就在哪。下單前看一眼籌碼卡。能不能當沖，就是當天買當天賣。有沒有被處置，就是被交易所管制。先排雷。進下單面板。價格我故意往下調幾檔。限價單就是到你指定的價才成交，調低它會掛著等。按第一下「買進下單」，按鈕會變成「確認買進」讓你核對。這是兩段式確認，再按一下才真正送出。別緊張，這裡是模擬環境，這筆是假錢。送出後切到「委託」分頁，單就掛在這裡，狀態、量、價一眼看完。要改價、改量、刪單，就在這排按鈕。最後示範一鍵清場。按一下 Esc，它先提醒你再按一次。零點六秒內補第二下，還沒成交的單全部撤掉。選股、確認、下單到刪單，這就是完整一輪。換你了。打開模擬環境，走熟這一遍，再談真錢。

### 錄製注意

全程模擬環境（徽章確認）。本章 Playwright 可全自動錄（`record-chapters.py` 的 chapter5）：排行複選→點列連動→籌碼卡→下單面板調價→兩段式送單→委託分頁→Esc×2 全刪。限價要先往下調幾檔讓單掛得住，Esc×2 才有東西可刪。非交易日錄影時排行榜可能空或靜態，建議交易日盤中重錄一次此章（腳本可重跑，成本為零）。
