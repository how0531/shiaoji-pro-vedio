## 剪映/CapCut 工作流（每一集同款）

1. 匯入該集 mp4 素材 → 拖到時間軸。
2. 「文字 → 文字朗讀」：貼上該章**旁白連續稿**，選台灣中文聲音（建議「知性女聲」或「沉穩男聲」，語速 1.0–1.1x）。
3. 對時間軸：素材片段依旁白節奏裁切/加速（操作等待段用 1.5–2x 加速，關鍵操作維持原速）。錄影日誌的場景時間戳可當下刀參考（`scripts/record-chapters.py` 執行時印出的 `[秒] ▶ 場景`）。
4. 「智慧字幕」自動產生字幕 → 校對專有名詞（Shioaji、TXFR1、ROD/IOC/FOK、OCO）。
5. 疊字卡：各段 caption 文字在腳本分鏡表「字幕」欄，直接複製。
6. 片頭 2 秒黑底標題卡＋片尾 CTA卡：「免費下載（模擬環境練熟再上）→ github.com/Sinotrade/shioaji-pro-app」。
7. 配樂：無版權 lo-fi/電子，音量 -25dB 以下不壓旁白。

## EP2 你的桌面版拍攝清單（照腳本第二章分鏡）

**開拍前**：⚠️ 用**假金鑰**示範輸入（如 `demo_api_key_xxxx`），或後製打碼；環境全程停在**模擬**；若會帶到瀏覽器的永豐 API 管理頁，帳號個資先打碼。

1. 🎬 桌面版**首次啟動**畫面（設定精靈）——如果你已登入過，從「伺服器」面板點「登出（清除 API 金鑰）」重回精靈
2. 🎬 點「選資料夾自動讀取 .env」示意（可略）
3. 🎬 在 API KEY／SECRET KEY 欄貼假金鑰 → 點眼睛鈕切明碼核對
4. 🎬 環境列：停在「模擬環境」→ 示範點一下「正式環境」讓紅字警告入鏡 → **切回模擬**
5. 🎬 點「啟動並開始使用」→ 拍到進度動畫與「約需 10–30 秒」提示 → 自動進入終端
6. 🎬 進終端後點頂部「伺服器」鈕，帶一圈浮層（狀態/環境切換/憑證/下單就緒檢查）
7. 每一鏡多留 3 秒頭尾，剪輯有餘裕；解析度 1920×1080、系統顯示縮放 100%

## 重錄素材（隨時可做）

```sh
# 模擬 server（1.7.0 binary 已在本機暫存資料夾）
cd C:\Users\How\AppData\Local\Temp\claude\shioaji-run
SJ_HTTP_ADDR=127.0.0.1:8080 sidecar-bin/shioaji.exe server start --no-open
# dev app（本機副本，OneDrive 版不要跑 pnpm）
VITE_API_TARGET=http://127.0.0.1:8080 node_modules/.bin/vite --port 5173
# 錄影（交易日盤中跑，行情回放段才有資料、閃動與五檔更有生命力）
PYTHONIOENCODING=utf-8 python scripts/record-chapters.py
python scripts/convert-and-frames.py
```

**建議在交易日盤中重錄一次 ch3／ch5**：行情回放會有資料、自選閃動與五檔跳動會讓畫面活很多。腳本重跑零成本。

## 🎬 評審 Agent 優化實施規範（後製必改）

依據小資新手、當沖老手與波段存股族的評審回饋，在剪映/CapCut 後製組裝時，必須嚴格執行以下細節優化：

### 1. 安全與風險警告字卡 (EP4 / EP2)
*   **K 線點價即下單警告 (EP4)**：在展示「點價買 / 點價賣」模式時，畫面中央必須疊加高對比度黃色字卡：
    > ⚠️ **點價下單警示**：本模式下，滑鼠單擊圖表價位將直接送出委託，系統**不會跳出 any 二次確認視窗**！
*   **客戶端觸價單失效警告 (EP4)**：在介紹圖上 OCO 停損停利虛線時，畫面必須疊加醒目的紅色驚嘆號字卡，且**至少停留 6 秒**（與旁白同步）：
    > 🚨 **本機監控聲明**：此括號單與圖上觸價皆由本機 App 執行。**若關閉程式或網路斷線，停損停利將完全失效！**
*   **明碼金鑰核對提示 (EP2)**：點擊眼睛核對金鑰時，底部加入小字浮水印：
    > ⚠️ **安全警告**：檢查完畢後，請務必再次點擊眼睛圖示，將金鑰遮蔽，避免個資外流。

### 2. 平台與專屬功能標註 (EP1 / EP2 / EP4)
*   **桌面版專屬標記**：在 `EP1` (介紹桌面與Web雙平台) 與 `EP2` (設定精靈/伺服器按鈕) 畫面邊角，常駐一行灰色小字標記：
    > 註：首次設定精靈與伺服器管理面板為桌面版 (Tauri) 專屬，開源 Web 版不支援。
*   **AI/回測限制標記**：在 `EP4` 展示 AI 副駕版面時，對著鎖定畫面標註：
    > 註：AI Agent 及策略回測為電腦版專屬閉源模組，Web 版將顯示鎖定畫面。

### 3. 操作手感與 HUD 疊加 (EP2 / EP4)
*   **鍵盤熱鍵 HUD 浮層**：在 `EP4` 示範「連按兩次 Esc 全刪單」以及「按 Esc 上鎖閃電下單」時，畫面中央或角落必須疊加一個橫置的 **Esc 鍵 HUD 動畫**，在按下按鍵的當下亮起，同步呈現系統反應。

### 4. 實戰淨收益解析 (EP5)
*   **真實賺賠框選**：在實戰結尾，旁白提到淨損益時，將畫面切至「損益/持倉」面板，並用**永豐紅 (#C0392B)** 粗框圈出「折讓後真實淨損益」的欄位，引導新手掌握交易成本。
