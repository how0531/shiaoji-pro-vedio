# AI Agent 特別集 · 設計（進階系列外掛）

> 使用者指定「特別設計一集」＋「完整做幾個實作」。實作已完成並實測可跑（見 `ai-agent-demos/`）。

## 先講清楚的製作限制

AI Agent 是 **Shioaji Pro 桌面版專屬的閉源模組**——開源 Web 版只會顯示 FeatureGate 鎖定畫面（「AI Agent 為桌面版專屬功能／下載桌面版」）。所以：
- **產品面板實際操作畫面**：Web 版錄不到，需**桌面版**錄（你有裝，或用官方素材）。
- **「AI Agent 能做什麼」的實料**：我用官方 Shioaji SDK 寫了**三個真的能跑的 agentic 交易代理**當作這集的核心示範——這正是那個面板背後在做的事。

## 這一集的內容結構（6 段）

| # | 畫面 | 旁白重點 |
|---|---|---|
| ai-1 | 片頭卡「AI Agent」＋大綱卡 | 這集講 AI 交易代理：它能自己抓資料、做決策、幫你執行；並看三個真實範例 |
| ai-2 | Web 版開「AI Agent」面板 → FeatureGate 鎖定畫面（**Web 可錄**） | 誠實說明：AI Agent 是桌面版專屬功能，Web 版看到的是鎖定畫面，要用請下載桌面版；它支援多供應商（Claude／Codex）agentic 對話、技能市集、排程任務、從你的操作觀察學習 |
| ai-3 | 終端跑 `agent1_premarket_scan.py`（**終端錄影**／或輸出動態卡） | 實作一：盤前掃描播報。抓漲幅/量/額三榜、交叉篩量價俱足、產出口語播報與自選候選——一句指令，開盤前功課做完 |
| ai-4 | 終端跑 `agent2_bracket_agent.py 2317 --demo` | 實作二：括號單自動風控。進場成交後自動掛 OCO 停損停利、客戶端監看到價才送單——保護單交給代理盯 |
| ai-5 | 終端跑 `agent3_position_monitor.py --once` | 實作三：持倉監控 Kill-Switch。盯當日損益，超過日虧上限自動撤單示警——情緒失控前先幫你踩剎車 |
| ai-6 | 桌面版 AI Agent 對話框（**桌面版錄**）或結尾卡 | 這些邏輯在桌面版可以直接用對話驅動，或做成技能重複用；下載桌面版試試 |

## 三個實作（已完成、實測對模擬環境跑過）

`ai-agent-demos/`：`agent1_premarket_scan.py`、`agent2_bracket_agent.py`、`agent3_position_monitor.py`（＋`agent_common.py` 登入、`README.md`、實測輸出 `run1/2/3.txt`）。用官方 Shioaji Python SDK、`simulation=True`、帳號遮蔽。

## 怎麼把這集做出來

- **可 Web 自動錄**：ai-2 的 FeatureGate 鎖定畫面（用現成分段錄影管線，add_panel「AI Agent」即顯示鎖定畫面）。
- **終端錄影**（ai-3~ai-5）：這三段是跑 Python 代理，畫面是終端不是瀏覽器。做法二選一：
  1. 你用 OBS 錄終端執行過程（最真實；帳號輸出已遮蔽）；
  2. 我把 `run*.txt` 輸出做成**深色終端風格的動態字卡**（打字機效果），不需真錄終端。
- **桌面版面板**（ai-2 補充、ai-6）：需桌面版錄；沒有就用鎖定畫面＋結尾卡帶過。

## 建議

先把 **ai-2 鎖定畫面**用現成管線錄起來、**ai-3~ai-5 用終端動態卡**呈現三個實作——這樣整集不依賴桌面版就能完成，且「實作」是真的跑得出來的東西。桌面版面板畫面等你方便時補錄升級即可。
